EBT-Site3

EBT-Site3 ist eine Vorlage, um eine individuelle Webseite auf der Grundlage von EBT-Vue3 zu erstellen.

Um Ihre eigene Webseite zu erstellen:

  1. Loggen Sie sich auf Github ein
  2. Clonen Sie dieses Datendepot in ihr GitHub-Konto
  3. Passen Sie ebt-config.mjs an
  4. Geben Sie Ihre Änderungen auf GitHub ein
  5. Schalten Sie Github Pages ein

Das war's schon!
